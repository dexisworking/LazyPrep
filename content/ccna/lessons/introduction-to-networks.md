# Introduction to Networks

A **computer network** is a digital telecommunications network that allows nodes to share resources. In computer networks, computing devices exchange data with each other using connections (data links) between nodes.

## What is a Network?

At its simplest, a network consists of two or more computers connected together so they can share resources such as files, printers, or an internet connection.

```
+------------+               +------------+
|  Computer  | <===========> |  Computer  |
|     A      |   Data Link   |     B      |
+------------+               +------------+
```

## Key Components of a Network

Every network contains three major components:
1. **Devices**: End systems (PCs, servers, smartphones) and network intermediaries (routers, switches, firewalls).
2. **Media**: The physical cabling (copper, fiber optic) or wireless frequencies that carry data signals.
3. **Services**: Software applications or protocols running over the network (web servers, email clients).

| Component | Description | Examples |
|---|---|---|
| **End Devices** | Source or destination of message | PC, VoIP Phone, Server |
| **Intermediary Devices** | Provide connectivity and route data | Switch, Router, Firewall |
| **Network Media** | The pathway for transmission | Copper, Fiber Optic, Wireless |

## Network Types by Geography

Networks are classified based on their geographical size:

*   **LAN (Local Area Network)**: Connects nodes in a limited geographical area, such as a home, school, or single office building. High speed and low latency.
*   **WAN (Wide Area Network)**: Connects LANs across large geographical distances, such as countries or continents. Typically managed by service providers.
*   **MAN (Metropolitan Area Network)**: Spans a physical area larger than a LAN but smaller than a WAN, such as a city or university campus.
